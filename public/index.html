API Endpoints

User Endpoints

  • GET /users - Retrieve a list of all users
  • GET /users/:id - Retrieve details of a specific user by ID
  • POST /users - Create a new user
  • PUT /users/:id - Update an existing user
  • DELETE /users/:id - Delete a user

News Post Endpoints

  • GET /news - Retrieve a list of all news posts
  • GET /news/:id - Retrieve details of a specific news post by ID
  • POST /news - Create a new news post
  • PUT /news/:id - Update an existing news post
  • DELETE /news/:id - Delete a news post

Search and Pagination

  • GET /users?limit=number&offset=number - Retrieve users with pagination
  • GET /news?search=value - Search news posts by a specific field
